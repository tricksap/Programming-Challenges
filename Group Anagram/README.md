## Problem: Group Anagrams

Given an array of strings strs, group the 
anagrams
 together. You can return the answer in any order.

### Example:
#### Input 
["eat", "tea", "tan", "ate", "nat", "bat"] Output:

#### Output
```javascript
[
   ["eat", "tea", "ate"],
   ["tan", "nat"],
   ["bat"]
]
```
